# Introduction

Welcome to the official documentation for **Bunxyz**, a minimalistic and fast web framework for Bun, designed for simplicity and performance, with built-in support for Zod validation.

## What is Bunxyz?

Bunxyz is a web framework built on top of [Bun](https://bun.sh/), leveraging Bun's incredible speed and modern JavaScript features. It aims to provide a simple yet powerful foundation for building web applications and APIs, focusing on developer experience and performance. It integrates seamlessly with [Zod](https://zod.dev/) for robust data validation.
