# Introduction

Welcome to the official documentation for **Bunxyz**, a minimalistic and fast web framework for Bun, designed for simplicity and performance, with built-in support for Zod validation.

## What is Bunxyz?

Bunxyz is a web framework built on top of [Bun](https://bun.sh/), leveraging Bun's incredible speed and modern JavaScript features. It aims to provide a simple yet powerful foundation for building web applications and APIs, focusing on developer experience and performance. It integrates seamlessly with [Zod](https://zod.dev/) for robust data validation.

## Key Features

- 🚀 **Fast:** Built on Bun's high-performance runtime.
- ✨ **Simple API:** Minimalistic and intuitive API design.
- 📁 **File-Based Routing:** Optional convention-over-configuration routing based on your directory structure.
- ✍️ **Manual Routing:** Standard `app.get()`, `app.post()`, etc., methods available.
- 🛡️ **Middleware Support:** Easy integration of middleware functions.
- ✅ **Zod Validation:** First-class integration for validating request bodies, query parameters, and route parameters.
- 📦 **Custom Request/Response:** Extended `Request` and helper `Response` classes.
- 🔒 **TypeScript First:** Written in TypeScript with type safety in mind.
- 🛠️ **CLI Tool:** Includes a CLI for easy project scaffolding.
